- [React](https://pt-br.reactjs.org/)

### Installation

1. Clone the repo

```sh
git clone https://github.com/kyotodevIndie/product-view.git
```

2. Install the packages using NPM or Yanr

```sh
yarn
```

```sh
npm install
```

3. Happy coding!
